# 🛡️ Portal de Segurança - RailanTech Este é um projeto com **Qualidade Máxima e Custo-Benefício** para proteger pessoas contra golpes. ## Como funciona: 1. **Verificador de Links:** Compara sites com o banco de dados `sites.json`. 2. **Central de Relatos:** Envia histórias direto para a Inteligência Artificial através de um Webhook do Make. ## Como atualizar: * Para adicionar mais lojas seguras, basta digitar o link dentro do arquivo `sites.json`. * Para trocar a ligação com o robô, edite o link `webhookUrl` dentro do arquivo `reclamacoes.html`. Desenvolvido com estratégia e lógica por RailanTech. ♟️
